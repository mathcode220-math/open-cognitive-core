# Welcome to the OCCP Team! 🧠🚀

Thank you for being here! The **Open Cognitive Core Project (OCCP)** is a community-driven initiative, and we believe that anyone who wants to help building a local, independent AI chip should be able to do so—**without complex rules or headaches.**

Whether you are a professional hardware engineer, a software developer, or just someone who loves open-source technology, there is a place for you here.

---

## 👋 How to Start (Keep It Super Simple!)

You don't need to be an ASIC expert to contribute. Here are the easiest ways to get involved right now:

### 1. Simple Hardware Tasks
- Take our main matrix core (`systolic_array_param.sv`) and run it on your favorite simulator.
- Help us write simple math modules (like a basic `ReLU` activator or a simple adder).
- Check the code for any typos or improve the comments to make it easier for others to understand.

### 2. Software & Testing
- Write simple Python scripts to test our Verilog code outputs.
- Help us build better testbenches (`Testbenches`) to verify our designs.

### 3. Documentation & Ideas
- Improve our `README.md` or translate it into other languages.
- Share your ideas! Open a GitHub Issue to discuss how we can make the architecture better.

---

## 🛠️ The 3-Step Contribution Process

No complicated bureaucracy here. Just follow these three simple steps:

1. **Find Something to Do:** Look at our code, or check our open Issues. If you have an idea, just jump in!
2. **Make Your Changes:** Fork the repo, create a branch, and write your code or text.
3. **Send a Pull Request (PR):** Submit your PR, and we will review it together with a big thank you. 

---

## 💡 Code Guidelines (Just the Basics)
- Keep your code clean, readable, and well-commented.
- We use **SystemVerilog/Verilog** for hardware, and **Python** for testing/simulation.
- Treat everyone in the community with respect and kindness.

---

## ⚖️ Licensing
By contributing, you agree that your work will be open-source under the project's **CERN Open Hardware Licence (CERN-OHL-W)**, keeping this technology free for all humanity, forever.
